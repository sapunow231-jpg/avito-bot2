# Avito Telegram Bot (Render)

## Настройка на Render

1. Подключить репозиторий к Render → создать Web Service.
2. Добавить переменные окружения:
   - TOKEN
   - RENDER_EXTERNAL_URL
3. Start Command:
   ```
   python bot.py
   ```
4. Деплой через Git push или Manual Deploy.
5. В Telegram написать `/start` — бот ответит ✅
